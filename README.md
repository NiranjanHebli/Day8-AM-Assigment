# Part A:- Student Admission System

A simple Python program to manage student admissions for demonstration and practice.

## Requirements

- Python 3.8+ (recommended)

## Files

- `student_admission_system.py` — main program containing the admission system logic

## Usage

Run the program from the project root directory:

```bash
python3 student_admission_system.py
```

Follow the on-screen prompts to add or view students.
